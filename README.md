## 💭 История создания

> Сидел я однажды на работе, скучал от нечего делать.  
> И вдруг решил воплотить давнюю задумку — сделать живых виртуальных помощниц с характерами, голосами и душой.  
> Знаний поначалу было маловато, но на помощь пришёл **DeepSeek** — моя нейросеть-соавтор.  
> Мы вместе писали код, придумывали диалоги, настраивали голоса и собирали всё в единый проект.  
> **ChuMei Angels** — результат нашей совместной работы. Надеюсь, тебе понравится так же, как нам — создавать.

## ✨ Особенности

| Функция | Описание |
| :--- | :--- |
| 🎤 **Голосовое управление** | Команды через микрофон, распознавание речи |
| 🧠 **Локальный ИИ (Ollama)** | Модель `llama3.1:8b` работает офлайн |
| 🎭 **5 уникальных персонажей** | Чучу, Мэй, Хана, Ки, Симона — разные голоса и характеры |
| 📖 **История вечеринки** | Карусель из 5 голосов (Чучу → Мэй → Хана → Ки → Симона) |
| 🎲 **Случайные цепочки** | Дневные, ночные, криминальные и бытовые сценарии |
| 😴 **Режим сна** | Автоматически с 23:00 до 8:00, можно уложить командой |
| 🛡️ **Цензура / NSFW** | Два режима общения |
| 🎬 **Видео-аватар** | Анимация говорящего персонажа (idle/talking) |
| 🖥️ **Интерфейс Windows 11** | Тёмная тема, Mica-эффект, прогресс-бары отношений |

## 📦 Требования

### Системные
- **Windows 10/11**
- **Python 3.10 – 3.13**
- **Git** (для клонирования)
- **Микрофон** (для голосовых команд)

### Программные
- **Ollama** с моделью `llama3.1:8b` — [скачать](https://ollama.com/download)
- **Видео-файлы** в папке `assets/`:
  - `avatar_idle_1.mp4`
  - `avatar_talking.mp4`
